Dialog

new Dialog(config, opts)

Creates a new Dialog instance
Parameters:
Name Type Description
config object The configuration for the button.
Properties
Name Type Description
el HTMLElement The html element for the dialog.
titleEl HTMLElement html element for title of the dialog.
activeStateStyle String CSS class name reference for the dialogs active state, e.g. 'js-dialog--active'.
opts Object The configuration options for the dialog
Properties
Name Type Attributes Default Description
animate Boolean <optional>
 true Whether to animate the transiton.
transitionEndCallback Boolean <optional>
 The callback to call when the transition completes.
Source:
  • dialog.js, line 49
Example
// instantiate a new button
let dialog = new Dialog({
    el: '#main-dialog',
    titleEl: '#main-dialog-title',
    activeStateStyle: 'js-main-dialog--active'
}, {
    animate: true,
    transitionEndCallback() {
        console.log('The transition has completed');
    }
});
