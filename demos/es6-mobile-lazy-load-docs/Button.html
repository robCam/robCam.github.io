Class: Button

Button

new Button(config, opts)

Creates a new Button instance
Parameters:
Name Type Description
config object The configuration for the button.
Properties
Name Type Attributes Description
el HTMLElement The html element for the button.
hoverStateStyle String CSS class name reference, e.g. 'js-button--hover'.
activeStateStyle String <optional>
 CSS class name reference, e.g. 'js-button--active'.
hideStateStyle String <optional>
 CSS class name reference, e.g. 'js-button--hide'.
opts Object The configuration options for the button
Properties
Name Type Attributes Default Description
persistActiveState Boolean <optional>
 true Whether to persist the aciive state after the button have been pressed.
perimeterDelta Boolean <optional>
 40 Used to further extend the perimeter of the button bounds. The default unit is pixels.
onPointerEndCallback function <optional>
 A callback can be passed in at invocation which will be fired when the button is released.
Source:
  • button.js, line 53
Example
// instantiate a new button
let button = new RCAM.widgets.Button({
    el: '.button',
    hoverStateStyle: 'js-button--hover'
 }, {
    onPointerEndCallback() {
        console.log('the button fired!');
    }
});

Methods

destroy()

Removes all event listeners correlating to the instance.
The destroy method should be called prior to deleting the instance to allow the garbage collector to determine what is able to be reclaimed.
Source:
  • button.js, line 362
Example
// Create an instance
let button = new RCAM.widgets.Button({
    el: '.button',
    hoverStateStyle: 'js-button--active'
});
// Destroy an instance
button.destroy();
// Delete an instance by setting to null.
button = null;
// Or, delete an instance by setting to undefined.
button = undefined;
// Or, delete an instance via the delete operator.
delete window.button;
