Module: param

Device parameters module.
This module is used to normalise browser inconsistencies.

1. Exports the CSS vendor prefix as: cssVendorPrefix
2. Exports the JavaScript vendor prefix for transitionend as: transitionEnd
3. Exports a boolean value whether the device has touch or not as hasTouch
4. Exports the pointer start used (either touchstart or mousedown) as pointerStart
5. Exports the pointer move used (either touchmove or mousemove) as pointerMove
6. Exports the pointer end used (either touchend or mouseup) as pointerEnd
7. Exports either touchcancel as pointerCancel
8. Exports either the orientationchange or resize event as resizeEvent
